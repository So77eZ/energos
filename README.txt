⚡ EnergyRank — сайт рейтинга энергетических напитков

1. Назначение сайта

EnergyRank — это веб-платформа для анализа и сравнения энергетических напитков по вкусовым характеристикам, энергетическому эффекту и пользовательским отзывам.

Основная задача сайта:

помочь пользователю найти лучший энергетик

сравнить напитки по вкусу, цене и эффекту

визуализировать вкусовые характеристики

собрать пользовательские отзывы

Сайт сочетает:

каталог напитков

визуальную аналитику вкусов

систему отзывов

2. Основная концепция

Каждый энергетический напиток имеет:

название

цену

фотографию

вкусовые характеристики

энергетические характеристики

флаг без сахара

пользовательские отзывы

Все напитки отображаются в плиточном каталоге.

8. Рейтинг напитка

Рейтинг рассчитывается автоматически на основе критериев.

Каждый критерий оценивается:

от 1 до 5

Критерии:

кислотность

сладость

газированность

насыщенность вкуса

послевкусие

уровень энергии

ценность (цена/качество)

Среднее значение преобразуется в:

⭐ звёздочный рейтинг

На странице присутствует 2D карта вкусов.

Страница отзывов открывается для конкретного напитка.

15. Система отзывов

Каждый отзыв содержит:

16. Добавление отзывов

Пользователь может добавить отзыв.

Форма содержит:

После отправки отзыв:

17. Удаление отзывов

Администратор может удалить отзыв.

Кнопка:

удалить

После удаления отзыв исчезает из списка.

18. Редактирование отзывов

Редактировать можно только свои отзывы.

Ограничения:

нельзя редактировать чужие отзывы

нельзя менять оценки старых отзывов

Это предотвращает:

манипуляции рейтингом
19. Хранение данных
