# Замечания и предложения по проекту Energos

## Актуальные задачи

### 🔴 Высокий приоритет

- **Убрать из админки оставление отзывов** — администратор должен оставлять отзывы через обычный интерфейс, а не отдельную форму в панели управления.

### 🟡 Средний приоритет

- **Сортировка по цене** — в фильтре не хватает сортировки по дороговизне.
- **Сортировка и фильтрация отзывов** по рейтингу и дате.
- **Расширенный фильтр** — всплывающее окно с фильтрацией по параметрам напитка (сладость, кислотность и т.д.), а не только сортировка.
- **Полнотекстовый поиск** — нечёткий поиск через `pg_trgm` (монстер / монстэр / monster).
- **Добавить способ обратной связи** — ссылка на тг-бота или личку.
- **Карта вкусов: выбор осей X/Y** — сейчас отображаются только 2 метрики из 6.

### 🟢 Низкий приоритет / По желанию

- **Заявки на добавление энергетиков** — форма с названием + опциональной ссылкой на референс.
- **Сравнение напитков** — режим side-by-side по всем метрикам.
- **Сброс пароля / верификация email** — стандарт для пользовательских платформ.
- **Кэш на фронте** — React Query или SWR вместо прямых fetch.
- **Англо-русский словарь** — «бэрн» → предлагать «Burn» при схожести >90%.
- **Парсинг актуальных цен** — внешний источник данных.
- **Вебсокет для обновления цен** — по приколу, real-time цены.
